# Form

HTML form manager. 

## Install

```
yarn add @arturdoruch/form
```

## Contents

Form class methods:

 * getName
 * getAction
 * getMethod
 * getElement
 * removeElement
 * submit
 * addElementListener
 * addSubmitListener
 * createRequestUrl
 * getData
 * resetData
 * setData

## Usage

```js
import Form from '@arturdoruch/form';
```
